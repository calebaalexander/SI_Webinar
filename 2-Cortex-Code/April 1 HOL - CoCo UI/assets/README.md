This folder is for quickstart guide images (screenshots, diagrams).

When submitting to sfguides, add screenshots here:
- overview_pipeline.png (pipeline diagram from conclusion)
- coco_panel.png (Cortex Code side panel)
- fix_button.png (inline Fix button on error)
- notebook_cells.png (notebook with multiple analysis cells)
- dynamic_table_results.png (DT query results)
- streamlit_dashboard.png (deployed Streamlit app)
- intelligence_agent.png (Snowflake Intelligence conversation)

Image guidelines:
- Lowercase filenames with underscores
- No special characters
- Under 1MB per image (optimize with tinypng)
- Do not create subfolders
